# 06 Servo Tarama Testi

Bu klasörde L293D Motor Shield üzerindeki SERVO1 pinine bağlı servo motoru test etmek için kullanılan Arduino kodu bulunur.

Bu test kodu servo motoru 20 derece ile 160 derece arasında sürekli sağa-sola hareket ettirir.

## Bağlantı

| Servo Kablosu | Motor Shield SERVO1 |
|---|---|
| Kahverengi / Siyah | - |
| Kırmızı | + |
| Turuncu / Sarı | S |

## Kullanılan Kod

sketch_may21d_servo_tarama_test_20_160.ino

## Amaç

Servo motorun doğru bağlandığını ve 20–160 derece aralığında tarama hareketi yapabildiğini kontrol etmek.
