# 02 Bluetooth Terminal Komut

Bu klasörde Serial Bluetooth Terminal uygulaması ile F, B, L, R ve S karakterleri gönderilerek robot arabanın kontrol edildiği Arduino kodu bulunur.

# Arduino L293D 4WD Robot Kodları

Bu depoda Arduino Uno ve L293D Motor Shield kullanılarak geliştirilen 4WD Bluetooth robot araba kodları bulunmaktadır.

## Klasörler

| Klasör | Açıklama |
|---|---|
| 01_motor_test | Motorları tek tek test etmek için kullanılan ilk motor test kodları |
| 02_bluetooth_terminal_komut | Serial Bluetooth Terminal ile F, B, L, R, S karakterleri göndererek robotu yönetme kodu |
| kutuphane | L293D Motor Shield için gerekli Adafruit Motor Shield kütüphanesi |

## Kullanılan Komutlar

| Komut | Görev |
|---|---|
| F | İleri |
| B | Geri |
| L | Sol |
| R | Sağ |
| S | Dur |
